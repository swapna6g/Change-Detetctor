Block 3 — Document Parsing
Prompt for Step 6: Build the document parsers
Create a new file backend/parsers.py with the following 
parsing functions:

1. parse_pdf(filepath) -> str:
   Use pdfplumber to open the PDF.
   Extract text from every page.
   Join pages with double newline separator.
   If extracted text is empty or under 100 characters
   return a dict: 
   {"error": "scanned_pdf", "text": ""}
   On success return: {"error": None, "text": full_text}
   Wrap in try/except returning error dict on failure.

2. parse_pptx(filepath) -> dict:
   Use python-pptx to open the file.
   Iterate through all slides in order.
   For each slide:
     - Get the slide title if it exists and mark it as:
       "=== SLIDE {number}: {title} ==="
     - Extract text from all shapes that have text frames
     - For tables on the slide extract each row as 
       pipe-separated values: "col1 | col2 | col3"
   Join all content with newlines.
   Return: {"error": None, "text": full_text}
   Wrap in try/except returning error dict on failure.

3. parse_docx(filepath) -> dict:
   Use python-docx to open the file.
   Iterate through all paragraphs.
   For paragraphs with heading style prefix with "## "
   For regular paragraphs use text as-is.
   Skip empty paragraphs.
   Join with newlines.
   Return: {"error": None, "text": full_text}
   Wrap in try/except returning error dict on failure.

4. parse_ucdd_excel(filepath) -> dict:
   Use openpyxl to open the file.
   Try to find a sheet named "Use Case Dictionary"
   If not found use the first sheet.
   Read row 1 and row 2 to extract Use Case Identifier
   and Use Case Name from the merged header area.
   Read row 4 as the column headers list.
   Read all rows from row 10 onwards as data rows.
   Skip completely empty rows.
   Convert each data row to a dictionary using 
   column headers as keys.
   Return:
   {
     "error": None,
     "use_case_identifier": value or "",
     "use_case_name": value or "",
     "ucde_rows": [list of row dictionaries],
     "column_headers": [list of header strings]
   }
   Wrap in try/except returning error dict on failure.

5. parse_document(filepath, doc_type) -> dict:
   doc_type is "report" or "ucdd"
   Get file extension from filepath (lowercase).
   If doc_type is "ucdd": call parse_ucdd_excel()
   If extension is ".pdf": call parse_pdf()
   If extension is ".pptx": call parse_pptx()
   If extension in [".docx", ".doc"]: call parse_docx()
   If extension is ".txt": read file directly as text
     and return {"error": None, "text": content}
   If extension not recognized return:
   {"error": "unsupported_format", "text": ""}

6. clean_placeholders(text) -> tuple:
   Use re.sub to replace all occurrences of 
   ##+ (two or more hash symbols followed by 
   optional digits) with "[REDACTED]"
   Return tuple: (cleaned_text, replacement_count)
   Import re at the top of the file.

Add this import at the top:

from pptx.util import Inches
import docx
import openpyxl
from pathlib import Path
